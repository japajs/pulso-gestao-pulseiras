# Meu Primeiro Projeto Web - Pulso

Este é um projeto simples de controle de estoque de pulseiras, contendo HTML, CSS e JavaScript.

## Arquivos

- `índice.html`: A estrutura da página.
- `índice.css`: O estilo da página.
